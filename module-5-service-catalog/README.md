# Service Catalog & Self-Service in ServiceNow

**Module objective:** Build self-service request experiences for end users with catalogs and record producers.

## Lessons in this module

* [5.1 — Create a Service Catalog Using Catalog Builder](lesson-5-1-create-a-service-catalog-using-catalog-builder.md)
* [5.2 — Order Items Using the Service Catalog](lesson-5-2-order-items-using-the-service-catalog.md)
* [5.3 — How to Create a Record Producer](lesson-5-3-how-to-create-a-record-producer.md)
* [5.4 — Hide Fields Using Catalog UI Policies](lesson-5-4-hide-fields-using-catalog-ui-policies.md)

## What you'll be able to do

* Use Catalog Builder to create a catalog item
* Define variables (questions) on a catalog item
* Publish a catalog item for ordering
* Order an item from the catalog as an end user
* Track the resulting request and request item
* Understand the request fulfillment flow
