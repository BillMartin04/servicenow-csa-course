# Exam 2 — Platform UI Configuration

**Quick answer:** Record a single video of **5 minutes maximum** demonstrating the tasks below in your own ServiceNow Personal Developer Instance (PDI), narrating as you go. This exam covers **Platform UI & Configuration (Module 3)** and is worth **20 points**.

{% hint style="warning" %}
**Members-only review.** Your video is only graded if you are an active **Exam Navigator** member (or any higher tier) of [TechTalk with Bill](https://www.youtube.com/channel/UCMf7pje1x5iZkkEF-Y3PLSA/join). Include your membership email/handle when you submit.
{% endhint %}

## The scenario

A team finds the Incident form cluttered and hard to use. You must reconfigure the list and form so it is cleaner and shows the right information.

## What you must demonstrate in your video

Complete and narrate each of the following:

1. Open any list (for example, **Incidents**) and **personalize/configure the columns** shown, then explain the difference between *Personalize* and *Configure*.
2. Open a form and use **Form Layout** (or **Configure > Form Layout**) to add or remove a field.
3. Use the **Form Designer** to rearrange fields, add a section, or add a field, and explain when you'd use Form Designer vs Form Layout.
4. Demonstrate the **Next Experience / Unified Navigation** — show how you navigate to an application or module.
5. Briefly explain how these UI changes are scoped (who sees them) and why that matters.

{% hint style="info" %}
**Time limit: 5 minutes.** Plan your steps before you hit record. Part of the skill is doing this efficiently — just like a real admin under deadline.
{% endhint %}

## Grading rubric (20 points)

| Criterion | What earns full marks | Points |
|-----------|----------------------|:------:|
| **List configuration** | Correctly configures list columns and explains Personalize vs Configure. | 4 |
| **Form Layout** | Successfully adds/removes a field using Form Layout. | 4 |
| **Form Designer** | Uses Form Designer correctly and explains when to use it vs Form Layout. | 5 |
| **Navigation fluency** | Demonstrates confident Next Experience / Unified Navigation. | 4 |
| **Clarity & narration** | Clear narration, organized, within 5 minutes. | 3 |
| **Total** | | **20** |

Your instructor will score each criterion and return written feedback. **You may resubmit** this exam if you score below the passing line — the goal is mastery and confidence.
