# Exam 3 — Data Foundations

**Quick answer:** Record a single video of **5 minutes maximum** demonstrating the tasks below in your own ServiceNow Personal Developer Instance (PDI), narrating as you go. This exam covers **Data Foundations: Tables, Modules & Update Sets (Module 4)** and is worth **20 points**.

{% hint style="warning" %}
**Members-only review.** Your video is only graded if you are an active **Exam Navigator** member (or any higher tier) of [TechTalk with Bill](https://www.youtube.com/@TechTalkwithBill/membership). Include your membership email/handle when you submit.
{% endhint %}

## The scenario

You need to build the data foundation for a new app and make sure your changes can be moved safely to another instance.

## What you must demonstrate in your video

Complete and narrate each of the following:

1. **Create an Update Set first** and set it as your current update set. Explain why you do this *before* making changes.
2. **Create a new custom table** (the 3-step way) with at least two custom fields.
3. **Create a module** in the Application Navigator that points to your new table.
4. Add **one or two records** to your table to show it works.
5. Open your update set and show that your changes were **captured**. Explain how you would move this update set to another instance.

{% hint style="info" %}
**Time limit: 5 minutes.** Plan your steps before you hit record. Part of the skill is doing this efficiently — just like a real admin under deadline.
{% endhint %}

## Grading rubric (20 points)

| Criterion | What earns full marks | Points |
|-----------|----------------------|:------:|
| **Update set discipline** | Creates and sets a current update set before changes; explains why. | 5 |
| **Table creation** | Creates a custom table with appropriate custom fields. | 4 |
| **Module creation** | Creates a working module pointing to the table. | 3 |
| **Data verification** | Adds records and confirms the table functions. | 3 |
| **Update set capture & migration** | Shows changes captured and explains migration to another instance. | 5 |
| **Total** | | **20** |

Your instructor will score each criterion and return written feedback. **You may resubmit** this exam if you score below the passing line — the goal is mastery and confidence.
