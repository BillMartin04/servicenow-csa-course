# Building Your First Application (Capstone) in ServiceNow

**Module objective:** Bring everything together by building a custom scoped application end to end.

## Lessons in this module

* [8.1 — Build Your First App — Part One](lesson-8-1-build-your-first-app-part-one.md)
* [8.2 — Build Your First App — Part Two](lesson-8-2-build-your-first-app-part-two.md)
* [8.3 — Build Your First App — Part Three](lesson-8-3-build-your-first-app-part-three.md)
* [8.4 — Create Dashboards in ServiceNow (2025 Update)](lesson-8-4-create-dashboards-in-servicenow-2025-update.md)
* [8.5 — Capstone Project & Next Steps](lesson-8-5-capstone-project-next-steps.md)

## What you'll be able to do

* Create a new scoped application
* Plan the app's tables and data model
* Set up the application scope correctly
* Build forms and lists for your app
* Add roles and access control to the app
* Configure the user experience
