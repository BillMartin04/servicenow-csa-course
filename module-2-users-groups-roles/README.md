# Users, Groups, Roles & Access Control in ServiceNow

**Module objective:** Manage who can access the platform and what they can do using users, groups, roles, and ACLs.

## Lessons in this module

* [2.1 — How to Create Users in ServiceNow](lesson-2-1-how-to-create-users-in-servicenow.md)
* [2.2 — How to Create User Groups](lesson-2-2-how-to-create-user-groups.md)
* [2.3 — How to Add Roles to a User Group](lesson-2-3-how-to-add-roles-to-a-user-group.md)
* [2.4 — ServiceNow ACLs & ACL Execution Hierarchy](lesson-2-4-servicenow-acls-acl-execution-hierarchy.md)

## What you'll be able to do

* Create a new user record as an administrator
* Populate the key fields on a user record
* Understand the role of the user table
* Create a user group
* Add members to a group
* Explain why groups simplify administration
